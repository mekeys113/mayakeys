MiniEELS: Learning-Based Snake Robot Locomotion
Autonomous Robots & Control Lab · California Institute of Technology

Developed reinforcement learning locomotion policies for a high-DOF, snake-like robotic platform inspired by JPL’s EELS robot, designed for autonomous navigation in cluttered, real-world environments.

Trained policies in MuJoCo with domain randomization over terrain, contact conditions, and friction parameters, emphasizing stability and continuous body–ground interaction.

Machine Learning Pipeline for Cassini CDA Mass Spectra
Jet Propulsion Laboratory (NASA JPL)

Developed a hybrid machine learning framework for analyzing time-of-flight mass spectra from the Cassini Cosmic Dust Analyzer, integrating a spectrum classifier, calibration regressor, and contrastive autoencoder.

Designed calibration-aware preprocessing pipelines incorporating stretch and time-offset parameters, enabling consistent alignment between synthetic and flight data. Implemented wavelet-based denoising and feature extraction to improve peak resolution and reduce false identifications across 50,000+ spectra.

Semantic Clustering for Automated Grading with Large Language Models
Adam Blank Research Group · California Institute of Technology

Designed an NLP-based grading platform for Caltech’s core computer science courses, leveraging MiniLM embeddings and density-based clustering (OPTICS) to group short-answer responses by semantic similarity.

Integrated large language models for labeling assistance and confidence scoring within a human-in-the-loop workflow, improving annotation quality and inter-grader agreement across multiple courses.

Computer Vision–Driven Morphogenesis Analysis in Caulerpa brachypus
Meyerowitz Lab · California Institute of Technology

Built a programmable light projection and imaging system using a DLP projector, Raspberry Pi, and high-resolution camera to study morphogenesis in a giant single-celled organism.

Implemented computer vision pipelines combining active contours, random forests, and the Segment Anything Model (SAM) to enable precise, adaptive analysis of organism growth and illumination feedback.

EEG-PreProp-RAG: Retrieval-Augmented Generation for EEG Preprocessing
Independent Project

Built an end-to-end retrieval-augmented generation system for correcting and generating EEG preprocessing code in Python, grounded in real-world MNE-Python workflows and expert-curated examples.

Implemented semantic retrieval, prompt chaining, and automated quality filters (regex and unit-test checks) to ensure safe, reproducible code outputs, prioritizing robustness and interpretability.

Multi-Label Chest X-Ray Classification on the CheXpert Dataset
Deep Learning Project

Developed and fine-tuned a ResNet-50–based deep learning framework for multi-label classification of thoracic pathologies using the CheXpert chest X-ray dataset.

Addressed label uncertainty and class imbalance through tailored loss formulations and evaluation metrics, achieving competitive performance and top-tier ranking within the course evaluation.
